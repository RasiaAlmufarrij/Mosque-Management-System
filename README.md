# Mosque-Management-System
Desktop application for managing mosque records using Python, Tkinter, SQLite, and Folium.

## Features

- Add new mosques
- Search by mosque name
- Display all records
- Update Imam information
- Delete mosques
- Display mosque location on an interactive map
- Suggest similar mosque names if the search is incorrect

## Technologies

- Python
- Tkinter
- SQLite
- Folium
- Object-Oriented Programming

## Database

The application stores:

- Mosque ID
- Mosque Name
- Mosque Type
- Address
- Coordinates
- Imam Name

## Map Feature

Mosques can be displayed on an interactive map generated using Folium.

## How to Run

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
